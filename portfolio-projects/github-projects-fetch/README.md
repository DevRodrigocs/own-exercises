## Features
- Real-time fetching with `fetch()` + `async/await`
- Full error handling (404, 403 rate-limit, network, empty repos)
- Loading & error states
- Fully responsive CSS Grid layout
- Search by username (Enter key support)
- Shows: name, description, language, stars, forks and direct link
- Modern UI with hover animations
